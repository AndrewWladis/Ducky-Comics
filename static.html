Static Shock will return to the DC Universe!

By Duck

Static Shock and other Milestone titles were recently revealed to be returning to the DC Universe. Three Milestone titles were revealed: Static Season One, Icon and Rocket Season One, and Hardware Season One. They all seem to be using the same ‘Season’ type series that The Green Lantern By Grant Morrison popularized. The first Milestone title; Static, will debut June 15th, 2021.
